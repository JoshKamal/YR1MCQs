The Anatomy module questions are now managed directly from the backend.

How to Add Questions:

Step 1: Edit the JavaScript File

Questions are stored in questions/anatomy_questions.js. Edit this file to add, remove, or modify questions.

Step 2: Follow the Required Format

Each question must follow the format shown below.

Step 3: Restart the Application

After making changes to the question files, restart the application for changes to take effect.

Where to Add Your Questions:

Open the file questions/anatomy_questions.js and edit the array of questions. Below is the location where you should add your questions:

// Anatomy Questions
const anatomyQuestions = [
  // EXISTING QUESTIONS WILL BE HERE
  
  // ADD YOUR NEW QUESTIONS HERE
  // Example format:
  {
    question: "Your new question text",
    options: [
      "A) Option 1",
      "B) Option 2",
      "C) Option 3",
      "D) Option 4",
      "E) Option 5"
    ],
    correctIndex: 2, // Index of the correct answer (0-based)
    explanations: [
      "Explanation for option A",
      "Explanation for option B",
      "Explanation for option C",
      "Explanation for option D",
      "Explanation for option E"
    ],
    slideLink: "Reference source",
    topic: "Topic name"
  },
  // You can add multiple questions separated by commas
];

Question Format:

{
  question: "What is the question text?",
  options: [
    "A) First option",
    "B) Second option",
    "C) Third option",
    "D) Fourth option",
    "E) Fifth option"
  ],
  correctIndex: 2,
  explanations: [
    "Explanation for why option A is incorrect",
    "Explanation for why option B is incorrect",
    "Explanation for why option C is correct",
    "Explanation for why option D is incorrect",
    "Explanation for why option E is incorrect"
  ],
  slideLink: "Reference source or slide link",
  topic: "Specific topic within Anatomy"
}

Important Notes:

  • Questions are now loaded directly from JavaScript files, not from localStorage
  • The editor interface has been removed to protect question integrity
  • All changes must be made in the source code files
